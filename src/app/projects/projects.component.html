Tuberculosis Detection from Chest X-Rays

This is my Final Year Project. This involves detection of Tuberculosis from the input chest x-ray images. We obtained the dataset from Kaggle. We train a Convolutional Neural Network (CNN) from the obtained X-Ray images. Also by applying various filters like texture based and Image-based we obtain the filtered images. We also use those images for training. By tuning the hyperparameters we able to obtain the Highest Effciency of 91.8%. We also provide a simplified GUI for prediction. User needs to upload the x-ray image. Our model will detect the presence of tuberculosis in the uploaded image with a probability percentage. It can generate a report in the form of Portable Document Format(pdf).

Technology Stack:

  • Python
  • Keras, fpdf
  • Django

Web-enabled GUI for tuning hyperparameters

This is my Seventh Semester Project. This involves creating a GUI for tuning Hyperparameters in Image Processing. First, the user needs to log in. After he/she can upload a dataset and choose hyperparameters like no of Convolutional Layers, Activation Function, No.of hidden layers, No.of Output Layers etc. Once chosen training will be carried out in the server side.
